Aplicación “Artículos Chedraui”


Descripción de desarrollo.
Esta aplicación servirá para la validación de artículos realizada por el área de factura electrónica, consiste en una aplicación grafica donde se pueden consultar los datos de los artículos registrados en SAP  y verificar que coincidan con la información enviada por los proveedores.

Justificación.
Antes del desarrollo de esta aplicación, la validación de datos se realizaban usando un desarrollo basado en Access, debido al volumen de información en los catálogos de artículos esta aplicación no podía gestionar de manera óptima estas validaciones.
Por tal  motivo se procedió a realizar una modernización de esta aplicación, usando una base de datos robusta y un lenguaje de programación formal.

Tecnología.
La aplicación “Artículos Chedraui” está desarrollada sobre lenguaje Visual Basic.Net usando el Framework 4.0, como base de datos usa MySQL 5.0 y para los reportes generados se usa la Librería ClosedXML.
Toda la tecnología usada para el desarrollo es de licencia de libre uso y distribución.
